## Задание 3.1: Стандартное отклонение цены продуктов

Скрипт для выполнения запроса

```sql
select 
product_type,
round(STDDEV(base_msrp)::numeric, 2) as "среднее отклонение цены"  -- округление для улучшения читаемости отчета
from products
group by product_type
order by "среднее отклонение цены" desc;                           -- для приоритизации категорий с высоким разбросом цен
```

**Комментарий:**
- **округление** для читаемости денежных значений
- **сортировка** для приоритизации категорий с высоким разбросом цен

## Задание 3.2: Количество уникальных клиентов в каждом штате

Скрипт для выполнения запроса

```sql
select 
state,
count(distinct customer_id) as unique_cust
from customers
where state is not null                         -- для повышения точности аналитики
group by state
order by state;
```

**Комментарий:**
- фильтр `state is not null` для исключения клиентов без региональной привязки и повышения точности аналитики

## Задание 3.3: Типы продуктов с максимальной ценой меньше 1000

Скрипт для выполнения запроса

```sql
select 
product_type,
max(base_msrp) as max_price
from products
group by product_type
having max(base_msrp) < 1000;
```
