# Cамостоятельная работа №2

## Цель
Научиться подключаться к базе данных PostgreSQL из Python, создавать схему и таблицы, наполнять их тестовыми данными, выполнять аналитические SQL-запросы и визуализировать полученные метрики в Yandex DataLens.

## Инструменты
- Python 3 (Google Colab)
- Библиотека `psycopg2`
- PostgreSQL
- Yandex DataLens
- Jupyter Notebook (.ipynb)
- Git (GitHub)

## Выполненные шаги

### 1. Подключение к базе данных
Установлено соединение с сервером PostgreSQL с использованием учётных данных. Проверка выполнена через запрос версии сервера. Использована обработка ошибок с `try-except-finally`, соединение закрыто корректно.

### 2. Создание схемы и таблиц
На основе ER-диаграммы (вариант HR) создана схема `hr`. В ней развёрнуты таблицы:
- `departments` (отделы)
- `position` (должности)
- `employees_hr` (сотрудники)
- `employment` (история трудоустройства)

Таблицы связаны по ключам, для `employees_hr` добавлены ограничения уникальности email.

### 3. Заполнение таблиц данными
В каждую таблицу внесены тестовые записи:
- 4 отдела,
- 5 должностей,
- 5 сотрудников с привязкой к отделам и должностям,
- история трудоустройства (5 записей).

Данные включают имена, даты найма, email, зарплаты.

### 4. Выполнение простого SELECT-запроса
Реализован запрос `SELECT * FROM employees_hr`, который вывел всех сотрудников с их полями. Результат отформатирован в виде таблицы.

### 5. Выполнение JOIN-запроса
Создан запрос, объединяющий `employees_hr`, `departments` и `position`, чтобы получить для каждого сотрудника:
- имя, фамилию, email,
- название отдела,
- название должности,
- зарплату.

Результат отсортирован по зарплате.

### 6. Сохранение результата в CSV и визуализация в Yandex DataLens
Результат JOIN-запроса выгружен в CSV-файл [employ_report.csv](employ_report_20260327.csv).  
На его основе в Yandex DataLens созданы:
- столбчатые диаграммы (количество сотрудников по отделам, средняя зарплата по отделам),
- круговая диаграмма (распределение сотрудников по должностям),
- таблица сотрудников,
- горизонтальная диаграмма зарплат от большего к меньшему.
