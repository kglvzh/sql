## Задание 5.1.1: найти продажи (sales) с суммой sales_amount = 0. (Индекс B-Tree). Анализ на сервере
***Задание изменено на:** оптимизация поиска крупных денежных транзакций для финансового мониторинга.*

**Причина и обоснование замены:** в таблице sales отсутствуют записи с sales_amount = 0 (результат запроса вернул 0 строк), что делает невозможным демонстрацию эффективности индекса. В рамках финансового мониторинга критически важным является контроль крупных денежных транзакций, которые могут требовать дополнительной проверки со стороны отдела комплаенс и риск-менеджмента. Поэтому условие было изменено на sales_amount BETWEEN 1000 AND 100000, что позволило на реальных данных оценить ускорение выборки с помощью B-Tree индекса.

Скрипт для выполнения запроса:
```sql
SELECT COUNT(*) as large_transactions
FROM sales 
WHERE sales_amount BETWEEN 1000 AND 100000;
```

## Задание 5.1.2: оптимизация поиска крупных денежных транзакций для финансового мониторинга. (Индекс B-Tree). Индекс локально

До индекса:
- тип сканирования: Seq Scan;
- реальное время: 2.400 ms;
- прочитано буферов: 369 (вся таблица);
- обработано строк: 35,119 отфильтровано + 2,592 результат = 37,711 всего;
- стоимость: 934.66.

Результат выполнения:

<img width="509" height="223" alt="image" src="https://github.com/user-attachments/assets/c2465f5e-7c19-49bf-8332-52a34b2236ca" />



Создание индекса:
```sql
CREATE INDEX idx_sales_amount ON sales (sales_amount);
```

После создания индекса:
- тип сканирования: Bitmap Index Scan → Bitmap Heap Scan;
- реальное время: 0.299 ms;
- прочитано буферов: 36 + 9 = 45 страниц (вместо 369);
- результат: 2,592 строки;
- ускорение: 2.400 / 0.299 = 8 раз быстрее.

Результат выполнения:

<img width="593" height="277" alt="image" src="https://github.com/user-attachments/assets/c5614416-e982-4226-87c3-29f859a23360" />


# Вывод
Применение B-Tree индекса на поле sales_amount ускорило выполнение запроса в 8 раз (с 2.4 мс до 0.3 мс) и сократило чтение буферов на 88% (с 369 до 45) благодаря переходу от Sequential Scan к Bitmap Index Scan.
